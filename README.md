## ЁЯЪА Getting Started

Want to run any project locally? Here's all you need:

**1. Clone the repo**
```bash
git clone https://github.com/wissamn19/50_python_projects.git
cd 50_python_projects
```

**2. Install dependencies**
```bash
pip install pandas plotly scikit-learn matplotlib seaborn jupyter
```

**3. Navigate to a project and launch the notebook**
```bash
cd youtube-channel-analysis
jupyter notebook
```

Each project folder contains its own dataset inside a `data/` subfolder тАФ no external downloads needed.
